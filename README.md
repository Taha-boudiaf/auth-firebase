#Technologies used

* React 
* Tailwind Css
* Firebase
#Installing dependencies
This repo uses Yarn and npm 
```
npm install 
yarn 
```
#Usage
Before you run the project, make sure to create a `.env.local` file ,add your `config` firebase 
To run the project:

```
npm run start
```
